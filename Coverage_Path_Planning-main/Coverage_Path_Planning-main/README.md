# UAV Coverage Path Planning with Endurance Constraint

> 基于续航约束的植保无人机喷洒路径优化与仿真系统  
> Harbin Institute of Technology — Undergraduate Thesis Project

---

## 项目简介

本项目实现了针对农田的无人机喷洒路径规划系统，核心特点是：

- 自动生成农田覆盖路径（Boustrophedon/牛耕式往返路径）  
- 考虑无人机续航约束，将完整路径拆分为多个飞行任务  
- 可视化路径，任务用不同颜色区分  
- 支持多组实验对比，便于毕业设计论文展示  

该系统适用于本科毕业设计、无人机农田喷洒模拟和路径优化研究。

---

## 功能特点

| 功能 | 描述 |
|------|------|
| 农田边界读取 | 从 CSV 文件加载农田边界数据 |
| 覆盖路径生成 | 根据作业宽度、重叠率、最小转弯半径生成最优路径 |
| 最优方向搜索 | 自动测试多个路径方向，选择总路径最短的方向 |
| 续航约束拆分 | 根据无人机最大航程将完整路径拆分为多个任务 |
| 可视化 | 输出路径图，任务用不同颜色区分 |
| 实验支持 | 支持修改无人机最大航程参数进行对比实验 |
